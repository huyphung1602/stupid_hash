# Stupid Hash
A gem that implement Stupid Hash Tables. It will run slower than Ruby Hash

## Install locally
Clone the git repo and run:
```
gem install stupid_hash
```

## Create the hash and set [key, value]
```ruby
hash = StupidHash::SHash.new
hash.set(key, value)
```

## Get value from key
```ruby
hash.get(key)
```
